# African Genius Academy Website
## Al-Falah International School, Ile-Ife, Osun State, Nigeria

### 📁 Project Files
- `index.html` — Main HTML file (open this in your browser)
- `style.css`  — All CSS styles
- `script.js`  — All JavaScript (lightbox, form, scroll reveal, nav)

### 🚀 How to Run Locally
1. Download all 3 files into the same folder
2. Double-click `index.html` to open in your browser
3. That's it! No server needed.

### 🌐 How to Deploy Online
1. Upload all 3 files to your web hosting (cPanel, Netlify, etc.)
2. Make sure `index.html`, `style.css`, `script.js` are in the same folder
3. Point your domain to `index.html`
